Documentation

Introdution

This is the documentation for the Dragon Ball Api and here we provide help on how to use it. Below, we will explore all the paths and data structures, helping you to get familiar with the API.

Routes

Base URL: https://www.dragonballapi.com/

The base URL is kept constant and it is essential to progressively add different paths to provide the desired information. All responses are in the form of GET requests via HTTPS and return data in JSON format.

Currently, there are one types of resources:

Characters: (Retrieves all the characters)

Dragon Ball: Currently, there are a total of 46 characters

Get all characters:

Dragon Ball: You can access the list of characters by using the /dragonball endpoint.

Dragon Ball Z: You can access the list of characters by using the /dragonballz endpoint.

Dragones: You can access the list of Dragons by using the /dragons endpoint.
